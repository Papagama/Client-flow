# ClientFlow — CRM для фрилансеров и агентств

ClientFlow — это CRM-система для фрилансеров и небольших агентств. Она помогает вести клиентов, проекты, задачи и оплаты в одном месте. Все данные хранятся в браузере, ничего не уходит на сервер.

## Запуск

```bash
cd react-app
npm install
npm run dev
```

Приложение откроется на http://localhost:5173/

## Начало работы

Откройте приложение. Вы попадёте на экран входа. Есть три варианта:

- **Зарегистрироваться** — нажмите «Зарегистрироваться», заполните имя, email и пароль. Аккаунт создастся и вы сразу попадёте в систему.
- **Войти через Google** — нажмите кнопку с логотипом Google. Система создаст аккаунт автоматически.
- **Демо-вход** — если хотите просто посмотреть, нажмите «Демо-вход». Загрузятся тестовые данные.

## Навигация

Слева — боковая панель с разделами:
- **Дашборд** — общая картина
- **Клиенты** — база клиентов
- **Проекты** — список проектов
- **Канбан** — доска задач
- **Настройки** — профиль, тема, бэкапы

Сверху — строка с названием раздела, поиском, колокольчиком уведомлений и аватаркой. На мобильных боковая панель скрыта, открывается по кнопке-бургеру.

## Быстрый поиск

Нажмите `Ctrl+K` в любом месте. Откроется палитра поиска. Начните вводить — система ищет по клиентам, проектам и задачам одновременно. Стрелками выберите результат, Enter — перейти.

## Уведомления

Колокольчик в правом верхнем углу показывает:
- Просроченные задачи (красные)
- Задачи с дедлайном в ближайшие 3 дня (жёлтые)
- Неоплаченные платежи (оранжевые)

Нажмите на уведомление — перейдёте к нужному проекту.

## Сценарий: от нового клиента до оплаты

Допустим, к вам обратился новый клиент — Иван из компании «Старт». Вот как это выглядит в ClientFlow:

### Шаг 1. Добавляем клиента

Перейдите в «Клиенты» → нажмите «+ Добавить клиента». Заполните:
- Имя: Иван Петров
- Контакт: ivan@start.ru
- Ниша: Стартапы
- Статус: Лид

Сохраните. Иван появится в таблице.

### Шаг 2. Создаём проект

Перейдите в «Проекты» → «+ Добавить проект»:
- Название: Лендинг для Старт
- Клиент: Иван Петров (выберите из списка)
- Этап: Бриф
- Дедлайн: через 2 недели
- Бюджет: 50000

### Шаг 3. Разбиваем на задачи

Нажмите на проект в таблице — откроется страница проекта. Вкладка «Задачи» → «+ Добавить задачу»:
- «Собрать ТЗ» — приоритет Высокий, срок через 3 дня
- «Дизайн макетов» — приоритет Высокий, срок через 7 дней
- «Вёрстка» — приоритет Средний, срок через 12 дней
- «Тестирование» — приоритет Низкий, срок через 14 дней

### Шаг 4. Работаем с канбан-доской

Перейдите в «Канбан». Все задачи в колонке «К выполнению». По мере работы перетаскивайте карточки:
- Начали собирать ТЗ → перетащите в «В процессе»
- Собрали → перетащите в «Готово»

Можно фильтровать по проекту через выпадающий список сверху.

### Шаг 5. Фиксируем оплаты

Вернитесь в проект → вкладка «Оплаты» → «+ Добавить оплату»:
- Аванс 25000 — статус «Оплачено»
- Финальная оплата 25000 — статус «Не оплачено»

Нажмите на статус оплаты, чтобы переключить: Не оплачено → Частично → Оплачено.

### Шаг 6. Добавляем ссылки

Вкладка «Ссылки» — добавьте ссылки на Figma, GitHub, документы. Всё под рукой.

### Шаг 7. Комментарии к задачам

На вкладке «Задачи» нажмите стрелку ▸ слева от задачи — раскроется секция комментариев. Пишите заметки, фиксируйте решения.

### Шаг 8. Меняем этапы

По мере продвижения проекта редактируйте его: Бриф → Дизайн → Вёрстка → Правки → Сдача. Прогресс-бар на странице проекта покажет процент выполненных задач.

### Шаг 9. Экспорт отчёта

Когда проект завершён, нажмите «Экспорт» на странице проекта — скачается JSON с полной информацией: задачи, оплаты, ссылки, итоги.

### Шаг 10. Дублирование проекта

Если у клиента похожий заказ — нажмите «Дублировать». Создастся копия проекта со всеми задачами, оплатами и ссылками.

## Дашборд

На главной странице видно:
- Сколько клиентов, проектов, задач и сумму оплат
- Ближайшие дедлайны (кликабельные — ведут к проекту)
- График задач за последние 7 дней

## Удаление с отменой

Когда вы удаляете клиента, проект, задачу, оплату или ссылку — внизу справа появляется уведомление с кнопкой «Отменить». У вас есть 5 секунд, чтобы вернуть удалённое.

## Настройки

В настройках (левая колонка):
- **Профиль** — измените имя и email
- **Тема** — переключите между светлой и тёмной
- **Экспорт/Импорт** — скачайте полный бэкап всех данных или восстановите из файла
- **Сброс** — вернуть всё к начальным тестовым данным

В правой колонке — статистика по записям, горячие клавиши и информация о приложении.

## Горячие клавиши

| Комбинация | Действие |
|---|---|
| `Ctrl+K` | Глобальный поиск |
| `Esc` | Закрыть модальное окно или поиск |
| `↑` `↓` | Навигация по результатам поиска |
| `Enter` | Открыть выбранный результат |

---

Это всё, что нужно для работы. Система простая: добавляете клиента, создаёте проект, разбиваете на задачи, отслеживаете оплаты. Всё остальное — фильтры, поиск, канбан, уведомления — помогает не терять ничего из виду.
